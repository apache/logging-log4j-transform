Log4j Transform CLI Tools

The Log4j Transform CLI Tools are a set of command line tools to help users with various Log4j-related formats.

log4j-codegen

The log4j-codegen tool is a small command line application that generates Logger wrappers with custom log levels.

Syntax

The general syntax of the tool is:

java -jar log4j-codegen-0.3.0-SNAPSHOT.jar [-f=<outputFile>] <command> <className> [<level>...]

where the elements have the following meaning:

<outputFile>

The name of the output file. By default, the output is printed on the standard output.

<command>

This can be one of:

extendedLogger

Generates a logger class that supports the standard log levels and some additional ones.

customLogger

Generates a logger class that supports only the levels specified on the command line.

<className>

The fully qualified name of the Logger class to generate.

<level>

A string of the form <levelName>=<levelPriority>, where:

<levelName>

The name of the level, e.g. CONFIG, AUDIT.

<levelPriority>

An integer that specifies the severity of each level. For a list of standard priorities see Levels.

Usage examples

java -jar log4j-codegen-0.3.0-SNAPSHOT.jar extendedLogger DIAG=350 NOTICE=450 VERBOSE=550
